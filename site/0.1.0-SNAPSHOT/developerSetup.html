Developer setup guide

These instructions are intended to assist the would-be AgileDev developer with setting up an environment in which to successfully build the AgileDev source code.

  • Maven

  1. Download and Install Maven - Download the latest version of Maven and install it

  2. Add username/password github account to settings.xml maven.

<servers>
  <server>
    <id>github</id>
    <username>GitHubLogin</username>
    <password>GitHubPassw0rd</password>
  </server>
</servers>

Once the above setup is complete:

  1. Open a fresh terminal
  2. Navigate to the `AgileDev directory, or whatever you called it.
  3. Type the following on the command line:
mvn clean install
