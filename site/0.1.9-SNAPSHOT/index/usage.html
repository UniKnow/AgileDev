Adding the plugin to your POM

The Maven Site Indexer is a standard maven plugin. Download the code, mvn install it and then add the following to your pom.xml 

 <plugin>
   <groupId>org.apache.maven.plugins</groupId>
   <artifactId>index</artifactId>
   <version>0.0.1-SNAPSHOT</version>
   <executions>
     <execution>
       <phase>site</phase>
       <goals>
         <goal>index</goal>
       </goals>
     </execution>
   </executions>
 </plugin>
