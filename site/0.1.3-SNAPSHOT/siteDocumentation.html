Site Documentation

All documentation of AgileDev is within the code base and is written in MarkDown. Markdown is a plain text formatting syntax designed so that it can be converted to HTML. There is no clearly defined Markdown standard but if you comply to the following syntax the generated pages should be formatted correctly.
